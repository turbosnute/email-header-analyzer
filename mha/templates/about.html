About Mail Header Analyzer

A tool to analyze email headers and track email delivery paths.

Original Development

Mail Header Analyzer (MHA) was originally created by Ahmed Shawky (@lnxg33k).

The original project is hosted on GitHub at https://github.com/lnxg33k/MHA

Features

  • Analyze email header metadata
  • Track email delivery paths
  • Calculate and visualize delivery delays
  • View security headers (SPF, DKIM, etc.)
  • Permanent links to saved analyses
  • Geolocate IP addresses in headers
